# 错题复习本 MVP v2

这是一个纯前端错题复习网站原型，可以直接在浏览器本地使用，也有 PWA 基础配置。

## 新增功能

- 智能复习算法：到期题、易错题、久未复习题、新题优先出现
- 类 Anki 间隔复习：答对后按 1 / 3 / 7 / 15 / 30 / 60 / 120 天延后
- 答错后立即重新进入待复习
- 每题记录连续答对次数、下次复习时间、复习历史
- 统计分析页：题库总数、今日应复习、已标熟、累计作答、总体正确率
- 最近 14 天刷题量趋势
- 分类题量和分类正确率
- 最需要回炉的错题排行

## 使用方法

直接打开 `index.html`，或在本文件夹运行：

```bash
python -m http.server 8000
```

然后访问：

```text
http://localhost:8000
```

## 数据保存

数据保存在浏览器 localStorage 中。建议定期在“备份/恢复”里导出 JSON。

## 注意

这是本地 MVP，不包含账号、多设备同步、云端图片备份。正式上线建议改成 React/Next.js + Supabase/Firebase。
